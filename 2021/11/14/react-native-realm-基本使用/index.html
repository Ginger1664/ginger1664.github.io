二、定义表结构

  1. 列类型说明
    • bool：映射到 JS 的 Boolean 类型。
    • int：映射到 JS 的 Number 类型，在 realm 中，以 64 bits 存储。
    • float：映射到 JS 的 Number 类型，在 realm 中，以 32 bits 存储。
    • double：映射到 JS 的 Number 类型，在 realm 中，以 64 bits 存储。
    • string：映射到 JS 的 String 类型。
    • decimal128：用于更高精度的数字。
    • objectId：映射到 BSON 的 ObjectId 类型。
    • data：映射到 JS 的 ArrayBuffer 类型。
    • date：映射到 JS 的 Date 类型。
    • list：映射到 JS 的 Array 类型，将 [] 加到类型后来指定为该原生类型的数组。
    • linkingObjects：用于定义逆关系的特殊类型。
    • dictionary：键值对。使用 {} 来定义类型。
    • set：集合类型，使用 数据类型<> 来定义。
    • mixed：可以用来存放不同的数据类型。
    • uuid：是来自 Realm.BSON 的通用唯一标识符。
  1. 定义可选类型：在类型后附加一个?。
  1. 指定逐渐：primaryKey:'_id'。
  2. 索引属性：指定索引会增加读的速度，减慢写的速度，使用 = 时最有效。
  1. 定义默认值：要定义默认值，请将属性值设置为具有type字段和default字段的对象 。
